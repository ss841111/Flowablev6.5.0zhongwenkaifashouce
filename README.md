# Flowable v6.5.0 中文开发手册

## 资源描述

本仓库提供 **Flowable v6.5.0 中文开发手册** 的下载资源。该手册是针对 Flowable 工作流引擎的详细中文文档，涵盖了 BPMN（业务流程建模与标注）的相关内容，适合开发人员、架构师以及对工作流引擎感兴趣的读者参考和学习。

## 手册内容概述

- **Flowable 简介**：介绍 Flowable 工作流引擎的基本概念和架构。
